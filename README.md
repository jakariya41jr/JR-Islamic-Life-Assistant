# # 🌙 JR Islamic Life Assistant (Digital Deen Portal)

**JR Islamic Life Assistant** হলো একটি আধুনিক, রেসপন্সিভ এবং ফিচার-সমৃদ্ধ প্রগতিশীল ওয়েব অ্যাপ্লিকেশন (PWA)। এটি মুসলিম ভাই-বোনদের দৈনন্দিন ইবাদত এবং ইসলামিক জীবনধারাকে সহজতর করার জন্য ডিজাইন করা হয়েছে।

## ✨ মূল বৈশিষ্ট্যসমূহ (Key Features)

- 🕋 **স্মার্ট ড্যাশবোর্ড:** ব্যবহারকারীর নাম এবং দৈনিক ইবাদতের সারাংশসহ পার্সোনালাইজড ইন্টারফেস।
- 🕒 **নামাজের সঠিক সময়সূচি:** Aladhan API ব্যবহার করে বিশ্বের যেকোনো প্রান্তের নামাজের সময় এবং পরবর্তী নামাজের কাউন্টডাউন।
- 🚫 **মাকরূহ সময় সতর্কবার্তা:** সূর্যোদয় ও সূর্যাস্তের নিষিদ্ধ সময় সম্পর্কে অটোমেটিক অ্যালার্ট।
- 📍 **মসজিদ ফাইন্ডার:** Leaflet.js এবং Overpass API ব্যবহার করে ব্যবহারকারীর বর্তমান অবস্থানের ২ কিমি এর মধ্যে থাকা সব মসজিদ ম্যাপে প্রদর্শন।
- 📿 **ডিজিটাল তাসবিহ:** হ্যাপটিক ফিডব্যাক (ভাইব্রেশন) এবং লোকাল স্টোরেজসহ ৩টি ক্যাটাগরির তাসবিহ কাউন্টার।
- 💰 **যাকাত ক্যালকুলেটর:** স্বর্ণ, রূপা, নগদ টাকা এবং ঋণের হিসাব অনুযায়ী নেসাব ভিত্তিক সঠিক যাকাত গণনা।
- 📖 **ইসলামিক লাইব্রেরি:** - আল্লাহর ৯৯টি নাম (অর্থ ও ফযীলতসহ)।
    - ২৫ জন নবীর সংক্ষিপ্ত জীবনী।
    - হিসনুল মুসলিম থেকে ৫০+ প্রয়োজনীয় দোয়া।
- 🎧 **অডিও কুরআন:** লাইভ কুরআন রেডিও এবং বিশ্বখ্যাত ক্বারীদের তিলাওয়াত শোনার ব্যবস্থা।
- 🧠 **ইসলামিক কুইজ:** ৩টি ভিন্ন লেভেলে (Beginner, Intermediate, Advanced) দ্বীনি জ্ঞান যাচাই।
- 🖼️ **আয়াত শেয়ারিং:** 'Ayah of the Day' জেনারেট করে সরাসরি ছবি হিসেবে ডাউনলোড করার সুবিধা।

## 🛠️ প্রযুক্তি (Technologies Used)

এই প্রজেক্টটি কোনো হেভি ফ্রেমওয়ার্ক ছাড়াই **Pure Vanilla JavaScript** ব্যবহার করে তৈরি করা হয়েছে যাতে এটি অত্যন্ত দ্রুত কাজ করে।

- **Frontend:** HTML5, CSS3 (Glassmorphism UI), Modern JavaScript (ES6+)
- **APIs:** Aladhan Prayer API, Overpass OpenStreetMap API, AlQuran Cloud API
- **Libraries:** Leaflet.js (Maps), html2canvas (Image Generation), FontAwesome (Icons)
- **Fonts:** Hind Siliguri (Bangla), Poppins (English)
- **Deployment:** PWA Ready (Service Worker Integration)
